## 两列等高的方法介绍

- border.html: 通过border来实现左右两边的等高布局
- max-padding: 通过大的padding-bottom，来讲容器撑的很大，然后再截断
- parent.html: 原理跟border.html差不多
- move.html: 将颜色全部放到父容器上面，然后通过位置的移动，来展示不同的背景色
- su-move.html: 将自适应跟两列自适应布局结合在一起
